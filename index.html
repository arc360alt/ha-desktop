Features
Everything you need,
nothing you don't

Built to feel native on every platform without any of the usual Electron bloat.

Persistent Sessions

Your login stays saved between launches. No more re-authenticating every time you open the app.

Custom Titlebar

A sleek frameless window with native-feeling controls that kinda matches the Home Assistant aesthetic.

Any Server

Connect to any Home Assistant instance, local, remote, or via Nabu Casa. Switch servers any time.

Developer Tools

Open Chromium DevTools directly from the settings menu for quick debugging and inspection.

Cross Platform

Available as a Windows installer and Linux AppImage or .deb package. One codebase, everywhere.
